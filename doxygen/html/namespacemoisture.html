const int moisture::getInterval ( const int numIntervals)

Berechnet das Intervall der Bodenfeuchtigkeit basierend auf der Kalibrierung.

Diese Funktion liest den aktuellen Bodenfeuchtigkeitswert von einem analogen Pin und berechnet das entsprechende Intervall basierend auf den Kalibrierungswerten für Luft und Wasser. Die Intervalle werden gleichmäßig zwischen den Kalibrierungswerten aufgeteilt.

Parameter
numIntervalsDie Anzahl der Intervalle, in die der Bereich zwischen den Kalibrierungswerten aufgeteilt werden soll.
Rückgabe
Das Intervall, in dem der aktuelle Bodenfeuchtigkeitswert liegt. Wenn der Wert außerhalb der Kalibrierungswerte liegt, wird das höchste Intervall zurückgegeben.

Definiert in Zeile 50 der Datei moisture.hpp.
