Testst Feuchtigkeit und triggert die Bewässerungssequenz.

Diese Funktion liest die Feuchtigkeitswerte von den analogen Pins und schaltet die Bewässerung ein, wenn der Wert über dem Schwellwert liegt. Die Funktion wird in der Hauptschleife aufgerufen.

◆ waterPlant()

void watering::waterPlant ( const int pinIndex)

Startet die Sequenz zum Gießen der Pflanze und wartet damit das Wasser versickern kann.

Diese Funktion aktiviert den DigitalPin und wartet 5 Sekunden, bevor sie ihn wieder deaktiviert. Danach wird 3 Minuten gewartet, bevor die Funktion beendet wird.

Parameter
pinIndexDer Index des DigitalPins, der aktiviert werden soll.
